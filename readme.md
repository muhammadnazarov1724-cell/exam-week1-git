# INNOMA.VC — Акселератор IT-бизнеса
Платформа для запуска технологического IT-бизнеса на международных рынках. Мы помогаем стартапам и инновационному бизнесу масштабироваться на рынки Ближнего Востока, Азии, Латинской Америки и Африки.
## Основные направления
- Аналитические исследования: Глубокий анализ технологических трендов на основе данных McKinsey, BCG, PWC, Deloitte и др.

- Онлайн акселератор: 8-недельная программа интенсивного обучения для подготовки проекта к презентации международным инвесторам.

- Сообщество экспертов: Доступ к профильным менторам и локальным партнерам по всему миру.
## Для кого проект?
1. IT-проекты на стадии идеи: Тестирование спроса и привлечение первых инвестиций.

2. Инновационный бизнес: Поиск новых рынков сбыта и масштабирование.

3. Корпорации: Внедрение внешних и внутренних инноваций.
## Программа акселерации
Программа включает 9 модулей, охватывающих все аспекты развития продукта:
- Модуль 1: Тенденции и IT-тренды.

- Модуль 2: Стартап-подход к продукту.

- Модуль 3: Поиск Product Market Fit.

- Модуль 4: Исследование целевой аудитории.

- Модуль 5: Создание MVP.

- Модуль 6: Unit-экономика.

- Модуль 7: Дорожная карта (Roadmap).

- Модуль 8: Работа с инвесторами.

- Модуль 9: Юридические аспекты и открытие юрлица.
## Технологии
Проект реализован с использованием классического веб-стека:

- HTML5: Семантическая разметка документа.

- CSS3: Адаптивная верстка (Flexbox, Grid), кастомные шрифты (Sans-serif).

- Responsive Design: Поддержка мобильных и десктопных устройств.
